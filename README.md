# Falcon 9 Launch Success Prediction

This project analyzes SpaceX Falcon 9 launch data to understand the factors that influence landing success and to build a predictive machine learning model.

The project covers the full data science workflow: data collection, web scraping, data wrangling, exploratory data analysis, interactive visualization, dashboard creation and predictive modeling.

## Project Structure

This project is intended to be explored in the following order:

1. [PedroPabloVicenteGonzález_Falcon9SuccessReport_ppt.pdf](./PedroPabloVicenteGonzález_Falcon9SuccessReport_ppt.pdf)  
   Final project presentation summarizing the analysis, methodology and results.

2. [Falcon9_SpaceX_API_calls_notebook.ipynb](./Falcon9_SpaceX_API_calls_notebook.ipynb)
   Data collection using the SpaceX API.

3. [Falcon9_TableScrap(Wiki).ipynb](./Falcon9_TableScrap(Wiki).ipynb)    
   Web scraping of Falcon 9 launch data from Wikipedia.

4. [Falcon9_DataWrangling.ipynb](./Falcon9_DataWrangling.ipynb)  
   Data cleaning, preparation and transformation.

5. [Falcon9_SQL(EDA).ipynb](./Falcon9_SQL(EDA).ipynb)  
   Exploratory data analysis using SQL queries.

6. [Falcon9_Visual_EDA.ipynb](./Falcon9_Visual_EDA.ipynb)  
   Visual exploratory data analysis using Python.

7. [Falcon9_Visual_EDA_Folium.ipynb](./Falcon9_Visual_EDA_Folium.ipynb)  
   Interactive map visualizations using Folium.

8. [Falcon9_Dash.ipynb](./Falcon9_Dash.ipynb)  
   Interactive dashboard creation.

9. [Falcon9_MachineLearningForPredictiveAnalysis.ipynb](./Falcon9_MachineLearningForPredictiveAnalysis.ipynb)  
   Machine learning models for predicting Falcon 9 first-stage landing success.

## Project Goal

The main goal of this project is to predict whether the Falcon 9 first stage will successfully land after launch.

This type of prediction is useful because Falcon 9 launches can be significantly cheaper when the first stage is recovered and reused.

## Repository Purpose

This repository contains the notebooks and final report developed as part of a data science project focused on SpaceX Falcon 9 launch success analysis.
